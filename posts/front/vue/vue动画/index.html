Vue过渡和动画#

1. 过渡和动画基础#

  1. 什么是过渡和动画

    1. 过渡：从一个状态向另一个状态变化的过程

    2. 动画：与过渡非常类似，只不过动画控制的过程更加细腻，更加精准

    3. 过渡语法：

      1
2
3
4

      <transition name="fade">
    <!-- 需要添加过渡的div标签-->
    <div></div>
</transition>

      name指定的是过渡所需要的样式名的前提

  2. transition组件

    1. Vue给transition标签，提供了六个过度类

      过渡类型 说明
      v-enter 进入过渡的开始状态，作用于开始的一帧
      v-enter-active 进入过渡生效时的状态，作用于整个过程
      v-enter-to 进入过渡的结束状态，作用于结束的一帧
      v-leave 离开过渡的开始状态，作用于开始的一帧
      v-leave-active 离开过渡生效时的状态，作用于整个过程
      v-leave-to 离开过渡的结束状态，作用于结束的一帧

    2. 过渡类分析图

    3. 代码示例：

    4. 总结

      1. v-enter 进入前的那一刻，和v-leave-to 离开结束的那一刻，就是隐藏的状态
      2. v-enter-to 进入结束的那一刻，和v-leave 离开前的那一刻，就是显示的状态
      3. v-enter-active 与 v-leave-active，是进入（显示）的整个过渡过程，与离开（隐藏）的整个过渡过程

  3. 自定义类名

    1. transition允许使用自定义的类名

    2. 如果使用自定义类名，就不需要设置name属性

    3. 设置自定义类名的属性

      1. 进入：enter-class、enter-active-class、enter-to-class
      2. 离开：leave-class、leave-active-class、leave-to-class

    4. 初始化渲染动画

            leave-active-class="animated bounceOutLeft">

</transition>

      上述代码中，appear表示开启此特性，appear-class表示初始class样式，appear-to-class表示过渡完成的class样式，appear-active-class会应用在整个过渡过程中

  5. 钩子函数实现动画

  6. Vue结合Velocity.js实现动画

    1. Velocity.js是一个简单易用，高性能且功能丰富的轻量级JS动画库

    2. 代码：

      3. 第三个参数可以配置duration动画时长，也可以配置动画完成complete

  3. 过渡模式

    1. 过度模式描述
      1. 新旧俩个元素参与过渡的时候，新元素的进入和旧元素的离开会同时触发，这是因为<transition>的默认行为进入和离开同时发生了，如果要求离开的元素完全消失后，进入的元素再显示出来（如开始的切换），可以使用transition提供的过渡模式mode，来解决当一个组件离开后，另一个组件进来时发生的位置的闪动或阻塞问题
      2. 过渡模式的原理时，设置有序的过渡而不是同时发生过渡，在transition中加入mode属性，它的俩个值如下所示。
      3. in-out：表示新元素先进行过渡，完成之后当前元素过渡离开
      4. out-in：表示当前元素先进行过渡，完成之后新元素过渡进入

4. 列表过渡#

  1. 什么是列表过渡

    通过v-for循环显示多个元素，并且通过transition-group组件来给这多个元素同时添加过渡效果

  2. 列表的进入和离开过渡

  3. 列表的排序过渡

    1. 排序过渡 - 其他元素平滑过渡

      当插入或移除元素的时候，虽然有过渡动画，但是周围的元素会瞬间移动到新的置，而不是平滑过渡。为了实现平滑过渡，可以借助v-move特性，v-move对于设置过渡的切换时机和过渡曲线非常有用。v-move特性会在元素改变定位的过程中应用，它同之前的类名一样，可以通过name属性来自定义前缀（例如name）

      1
2
3

      .list-move {
    transition: transform 1s;
}

    2. Flip

      Vue使用了FLIIP简单动画队列来实现排序过渡，所以即使没有插入或者移除元素，对于元素顺序的变化也支持过渡动画，FLIP动画能提高动画的流畅度，可以解决动画的卡顿，闪烁等不流畅的现象，它不禁可以实现单列过渡，也可以实现多维网络的过渡。FLIP代表First、Last、Invert、Play，有兴趣的读者可以自行研究学习

  4. 列表的交错过渡

    1. 列表的交错过渡，其实就是多个li在做动画时，每一个li做过渡时有时间差，依次过渡

                height: 0
