运算符【Go学习之路】#

运算符是一种特殊的符号，用以表示数据的运算、赋值和比较等（特别注意go语言里无三元运算符）

分类#

算术运算符#

用于对数值类型的变量进行运算

运算符 运算 范例 结果
+ 正号 +3 3
- 负号 -4 -4
+ 加 5 + 5 10
- 减 6 - 4 2
* 乘 3 * 4 12
/ 除 5 / 5 1
% 取模(取余) 7 % 5 2
++ 自增 a = 2 a++ a = 3
– 自减 a= 2 a– a = 1
+ 字符串拼接 “He” + “llo” “Hello”

  • 基本使用

    package main

import (
	"fmt"
)

func main() {
	//特别注意"/"、"%"、"++"、"--"

	// "/"的使用
	// 说明，如果运算的数都是整数，那么除后，去掉小数部分，保留整数部分
	fmt.Println(10 / 4) // 2

	var n1 float32 = 10 / 4
	fmt.Println(n1) // 2

	// 如果我们希望保留小数部分，则需要有浮点数参与运算
	var n2 float32 = 10.0 / 4
	fmt.Println(n2) // 2.5

	// "%"的使用
	// 取余公式 a % b = a - (a / b * b)
	fmt.Println("10 % 3 =", 10%3)     // 1
	fmt.Println("-10 % 3 =", -10%3)   // -1
	fmt.Println("10 % -3 =", 10%-3)   // 1
	fmt.Println("-10 % -3 =", -10%-3) //-1

	// ++ 和 --的使用
	var i int = 10
	i++                   // 等价i = i + 1
	fmt.Println("i =", i) // 11
	i--                   // 等价 i = i - 1
	fmt.Println("i =", i) // 10
}

  • 细节说明

    • 对于除号“/”，它的整数除和小数除是有区别的：整数除法，只保留整数部分而舍弃小数部分

    • 当对一个数取模时，可以等价a % b = a - (a / b * b)

    • Golang的设计者去掉C / Java中自增自减容易混淆的写法，让Golang更加简洁，统一。（强制性的，太搞笑了，估计是汤普森看不下C语言的八股文考试了）

      • Golang的自增自减只能当作一个独立语句使用，b := a++ 或者 b:=a –都是错的，在其他的语言里是被允许的
      • Golang的++和–只能写在变量的后面，不能写在变量的前面，++a，–a都是错的，在其他的语言里是被允许的
      • if i++ > 0 {xxx}的 写法也是不被允许的
      package main

import (
	"fmt"
)

func main() {
	var i int = 1
	// var b int = i++ //语法错误
	i++
	var b int = i
	fmt.Println("b =", b)
	// ++i //语法错误
	fmt.Println("b =", b)

	if i++ > 0 { //语法错误
		fmt.Println("i =", i)
	}
}

赋值运算符#

赋值运算符就是将某个运算后的值，赋给指定的变量

运算符 描述 示例
= 简单的赋值运算符，将一个表达式的值赋给一个左值 c = a + b，将a+b表达式结果赋给c
+= 相加后再赋值 c += a 等价于 c = c + a
-= 相减后再赋值 c -= a 等价于 c = c-a
*= 相乘后再赋值 c *= a 等价于 c = c * a
/= 相除后再赋值 c /= a 等价于 c = c / a
%= 求余后再赋值 c % a 等价于 c = c % a
«= 左移后赋值 c «= a 等价于c = c « a
»= 右移后赋值 c »= a等价于c = c » a
&= 按位与后赋值 c &= a等价于 c = c & a
^= 按位异或后赋值 c ^= a 等价于 c = c ^ a
!= 按位或后赋值 c != 2 等于c = c | 2

  • 基本使用

    package main

import "fmt"

func main() {
	// 赋值运算符的使用演示
	// var i int
	// i = 10 // 基本赋值

	// 有两个变量，a和b，要求将其进行交换，最终打印结果
	// a = 9, b = 2 == > a = 2, b = 9
	a, b := 9, 2
	fmt.Println("========交换前===========")
	fmt.Println("a =", a)
	fmt.Println("b =", b)
	t := a
	a = b
	b = t
	fmt.Println("========交换后===========")
	fmt.Println("a =", a)
	fmt.Println("b =", b)

	fmt.Println("======复合赋值的操作======")
	//复合赋值的操作
	a += 17 //等价于a = a + 17
	fmt.Println("a =", a)
}

// 结果
//========交换前===========
//a = 9
//b = 2
//========交换后===========
//a = 2
//b = 9
//======复合赋值的操作======
//a = 19

  • 细节说明

    • 运算顺序从右往左
    • 赋值运算符的左边，只能是变量，右边可以说变量、表达式、常量值

比较运算符/关系运算符#

关系运算符的结果都是bool类型，也就是要么是true，要么是false，关系表达式经常用在if结构的条件中或循环结构条件中

运算符 运算 范例 结果
== 相等 4==3 false
!= 不等于 4!=3 true
< 小于 4 < 3 false
> 大于 4 > 3 true
<= 小于等于 4 <= 3 false
>= 大于等于 4 >= 3 true

  • 基本使用

    package main

import "fmt"

func main() {
	var n1 int = 9
	var n2 int = 8
	fmt.Println(n1 == n2) // false
	fmt.Println(n1 != n2) // true
	fmt.Println(n1 > n2)  // true
	fmt.Println(n1 < n2)  // false
	fmt.Println(n1 >= n2) // true
	fmt.Println(n1 <= n2) // false

	flag := n1 > n2
	fmt.Println("flag =", flag) // true
}

  • 细节说明

    • 关系运算符的结果都是bool型，也就是要么true，要么是false
    • 关系运算符组成的表达式，我们称为"关系表达式", a > b
    • 比较运算符" == “不能写成 " = "

逻辑运算符#

连接多个条件(关系表达式)，最终结果也是一个bool

运算符 运算 范例 结果
&& 逻辑与运算符。操作符两边都为true则为true，反之则为false (1 == 1) && (2 == 2) true
|| 逻辑或运算符。操作符两边有一个true则为true，反之则为false (1 == 2) && (2 == 2) true
! 逻辑非运算符。如果条件为true，则逻辑为false，否则为true !(1 == 1) false

  • 基本使用

    package main

import "fmt"

func main() {
	// 演示逻辑运算符的使用 &&
	age := 40
	if age > 30 && age < 50 {
		fmt.Println("ok1")
	}

	if age > 30 && age < 40 {
		fmt.Println("ok1")
	}
	// ok1

	// 演示逻辑运算符的使用 ||
	if age > 30 || age < 50 {
		fmt.Println("ok3")
	}

	if age > 30 || age < 40 {
		fmt.Println("ok4")
	}
	// ok3
	// ok4

	// 演示逻辑非的使用 !
	if age > 30 {
		fmt.Println("ok5")
	}

	if !(age > 30) {
		fmt.Println("ok6")
	}
	//ok5
}

  • 细节说明

    • &&也叫短路与：如果第一个条件为false，则第二个条件不会判断，最终为false
    • ||也叫短路或：如果第一个条件为true，则第二个条件不会判断，最终结果为true

位运算符#

运算符 运算 范例
& 按位与 1 & 0 = 0
| 按位或 1 | 0 = 1
^ 按位异或 1^0 = 1
~ 按位取反 ~1 = 0

其他运算符#

运算符 描述 范例
& 返回变量存储的地址 &a；将给出变量的实际地址
* 指针变量 *a；是一个指针变量

  • 基本使用

    package main

import "fmt"

func main() {
	// 演示 & 和 *的使用（指针）

	a := 100
	fmt.Println("a的地址", &a) //0xc0000160b0

	var ptr *int = &a
	fmt.Println("ptr指向的值是", *ptr) //100
}

运算符优先级#

分类 描述 关联性
后缀 () [] -> . ++ – 左到右
单目 + - ! ~ (type) * & sizeof 右到左
乘、除、取余 * / % 左到右
加、减 + - 左到右
移位 « » 左到右
比较(关系) < <= >= > 左到右
相等(关系) == != 左到右
按位AND & 左到右
按位XOR ^ 左到右
按位OR | 左到右
逻辑AND && 左到右
逻辑OR || 左到右
赋值运算符 = += -= *= /= %= »= «= &= ^= != 右到左
逗号 , 左到右
