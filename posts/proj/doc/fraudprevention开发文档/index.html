用户模块#

登录#

  • url: /admin/login

  • method: Post

  • request body

    参数 说明 示例
    username 用户名 admin
    password 密码 admin
    token 验证码的token
    code 验证码 cwca1

  • response header

    参数 说明 示例
    Authorization 用户登录的token，其他的请求，请求头都要带上 xxxxx

    "code": 200,
    "msg": "success",
